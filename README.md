
# linux-cl

Linux-cl is a command-line tool for changing Linux tty coloud schemes. The success of the tool will vary from
terminal to terminal, but it works as expected on Ubuntu. Schemes taken from [iTerm2-Color-Schemes](https://github.com/mbadolato/iTerm2-Color-Schemes)
